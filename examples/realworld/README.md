# Real-World Examples

This directory contains real-world examples demonstrating JetORM usage in practical scenarios.

## Examples

### 1. E-Commerce System

**File:** `ecommerce_example.go`

Demonstrates:
- Product catalog management
- Order processing with transactions
- Inventory management
- Order history with pagination
- Low stock alerts

**Features:**
- Product CRUD operations
- Order creation with items
- Status management
- Category filtering
- Stock monitoring

### 2. Blog System

**File:** `blog_example.go`

Demonstrates:
- Blog post management
- Comment system
- Tag management
- Search functionality
- View tracking

**Features:**
- Post CRUD operations
- Comment moderation
- Published posts filtering
- Full-text search
- View counting

## Usage

These examples demonstrate:
- Complex entity relationships
- Transaction management
- Validation
- Pagination
- Search functionality
- Real-world patterns
